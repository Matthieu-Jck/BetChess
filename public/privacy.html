BetChess Privacy Policy

What BetChess collects

  • A lobby username so other players can challenge you.
  • Live match state such as moves, timers, invitations, and game results.
  • Technical connection data needed to run the multiplayer session.

What BetChess does not collect

  • No payment data.
  • No banking or real-money betting information.
  • No account password for gameplay, since the game currently uses guest usernames.

How data is used

Data is used only to show the lobby, run live matches, and deliver the game state between both players. Usernames are visible to other players in the lobby.
